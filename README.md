# ⭐ Sales Performance Analytics Dashboard | Excel Project

> Interactive Excel dashboard for analyzing sales executive performance, target achievement, and city-wise sales insights.

---

## 📌 Project Overview

This project focuses on analyzing **sales performance data** using Microsoft Excel to generate meaningful business insights related to sales executives, target achievement, and city-wise performance tracking.

The dashboard is designed to provide management with a quick overview of sales metrics, executive productivity, and performance gaps through interactive visualizations and filters.

This project demonstrates the use of **Excel as a business intelligence tool** for reporting and decision-making.
